# Orcs

A terminal-based autonomous orc clan simulator. Watch a clan of orcs live their daily lives — hunting for food, sleeping by the campfire, and wandering the land — all rendered in ASCII art.

## Run

```
cargo run
```

## Controls

| Key | Action |
|-----|--------|
| Space | Pause / Resume |
| +/- | Speed up / slow down |
| Arrows | Move cursor |
| Tab | Cycle selected orc |
| f | Drop food at cursor |
| q | Quit |
